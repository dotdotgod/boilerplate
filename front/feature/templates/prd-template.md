## 📡 API 명세

### 엔드포인트 목록
```
POST   /v1/[resource]           # 생성
GET    /v1/[resource]           # 목록 조회
GET    /v1/[resource]/:id       # 단일 조회
PUT    /v1/[resource]/:id       # 수정
DELETE /v1/[resource]/:id       # 삭제
```

### 요청/응답 형식
```typescript
// 요청
interface [Action]Request {
  field: string;
}

// 응답
interface [Action]Response {
  success: boolean;
  data: any;
  message: string;
}
```

### 에러 처리
```typescript
interface ErrorResponse {
  success: false;
  error: {
    code: string;
    message: string;
    details?: any;
  };
}
```
